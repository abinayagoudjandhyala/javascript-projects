
# JavaScript Projects 

This repository contains a collection of **JavaScript projects built while learning core JavaScript concepts**.
Each project focuses on a specific concept such as **Control Structures, Functions, DOM manipulation, and Asynchronous JavaScript**.

The goal of this repository is to **strengthen my JavaScript fundamentals by building practical mini-projects and interactive web applications**.

#  Concepts Covered

This repository demonstrates my understanding of important JavaScript concepts:

* Control Structures (Loops & Conditions)
* Functions
* DOM Manipulation
* Event Handling
* Asynchronous JavaScript
* Interactive Web Applications

By building these projects, I improved my **problem-solving skills and frontend development knowledge**.
